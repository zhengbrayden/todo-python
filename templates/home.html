Welcome to Todo API

This is a RESTful API for managing personal to-do lists with user authentication and CRUD functionality.

Available Endpoints:

POST /api/register/

Register a new user account

POST /api/login/

Login to get access token

GET /api/todos/

Get all todos (requires authentication)

POST /api/todos/

Create a new todo (requires authentication)

PUT /api/todos/:id/

Update a todo (requires authentication)

DELETE /api/todos/:id/

Delete a todo (requires authentication)
